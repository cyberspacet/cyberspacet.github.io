La transattention omniprésente : le deep web

Le deep web (ou web invisible) est une partie du web qui n’est pas indexée par les moteurs de recherche. A ne pas confondre avec le dark web (ou dark net), le deep web possède du contenu indexable “non choquant”.

Les raisons pour lesquelles ce contenu n’est pas indexé sont diverses : pages nécessitant un paiement, comptes bancaires, emails, contenu réservé à des abonnés (exemple d’un article de presse payant), arborescence trop complexe, etc.

source

Nous accédons donc au deep web tous les jours, seulement nous ne nous en rendons pas compte. Le cyberespace transattentionnel englobe cette partie de la toile : plutôt que de rester sur le web surfacique, qui est la partie du web la plus connue avec tous les éléments indexés, nous faisons preuve de transattention pour aller au-delà de ce que l’on peut voir et de ce que l’on connaît. Cette notion est donc omniprésente dans notre quotidien, et nous fait utiliser la notion de transattention inconsciemment.
Mais le cyberespace transattentionnel peut aussi être utilisé intentionnellement pour accéder à des endroits plus sombres d’internet, comme le dark net.
